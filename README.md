# AI Product Research Agent

Autonomous AI workflow that converts Reddit and Hacker News discussions into structured Product Requirement Documents (PRDs).

## Tech Stack
- n8n
- Ollama
- Phi 3
- REST APIs
- JavaScript

## Features
- Reddit + Hacker News scraping
- AI-powered pain point analysis
- Competitor detection
- PRD generation
- Markdown export

## Files Included
- Workflow JSON
- Documentation PDF
- Sample PRD outputs
